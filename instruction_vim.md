# VIM

## vim의 기본적인 명령어

**vim**	시작

**:q!**	저장하지 않고 끝내기

**:q**		저장하고 끝내기

**:wp**	저장하고 끝내기

**====================================**

>vim 내부에서 사용

**i**		텍스트 삽입

**x** 		텍스트 삭제

**h j k l**		차례대로 왼쪽 아래 위 오른쪽 으로의 이동
